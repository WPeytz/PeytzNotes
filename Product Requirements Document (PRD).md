1. Product Requirements Document (PRD)

Project name (working)

PeytzNotes

One-liner

An AI system that turns 5 years of university notes into a searchable, connected, personalized study assistant.

Core user (you)
	•	DTU student
	•	Has notes across multiple courses
	•	Needs revision, synthesis, and cross-course understanding

Core jobs to be done
	•	Find information quickly across all notes
	•	Understand concepts across courses
	•	Generate summaries for exams
	•	Turn notes into flashcards
	•	Reuse past knowledge

⸻

MVP features (strict scope)

1. Import Notion notes
	•	Upload exported Notion markdown/html
	•	Parse into structured format
	•	Assign metadata (course, year)

2. Search
	•	Semantic + keyword search
	•	Filter by course

3. Chat with notes
	•	Ask questions
	•	Answers must cite sources
	•	Show which notes were used

4. Exam summary
	•	Select course
	•	Generate structured summary

5. Flashcards
	•	Generate Q/A pairs from notes

⸻

Non-goals (important)
	•	No model training
	•	No multi-user SaaS
	•	No mobile app (yet)
	•	No complex agents
